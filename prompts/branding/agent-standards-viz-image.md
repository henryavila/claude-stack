# Agent Standards — Viz Image Prompt

Use este prompt para gerar a imagem principal de branding do produto **Agent Standards** em `viz-image` / ComfyUI.

## Primary Prompt

```text
Create a premium brand key visual for a developer tool called "Agent Standards".

What must be communicated:
- this is not a stack-first product
- this is a universal operational quality standard for AI agents
- the core is structured, disciplined, precise, and trustworthy
- stack modules are optional extensions, secondary to the core
- the product feels premium, technical, mature, and deliberate

Visual concept:
Design an elegant hero image that feels like a fusion of a luxury technical manual, a standards system, and a modern software brand. The image should center on the idea of structured operational quality for AI agents, not robots, not sci-fi fantasy, not generic developer neon.

Show an abstract visual system built from:
- a central "core" block or slab that feels foundational, engineered, and calm
- a few secondary modular tiles or panels orbiting it, clearly subordinate, representing optional stack modules
- precise alignment lines, indexing marks, registration ticks, spacing grids, and subtle document-system cues
- layered materials that imply operational rigor: brushed graphite metal, matte ceramic, smoked glass, heavyweight paper, fine embossing, satin-coated surfaces
- restrained signals of intelligence and automation through structure and flow, not through robot faces or circuit brains

Composition:
- premium centered composition
- central object dominant, modules secondary
- symmetrical or near-symmetrical layout with disciplined grid logic
- enough empty space to feel confident and expensive
- subtle depth, crisp layering, and intentional hierarchy
- image should feel like a launch visual for a serious high-end software tool

Symbolic direction:
- the core should look like a standards engine, a system of order, a calibrated operating layer
- modules should look attachable, optional, and neatly integrated
- visual language should imply "rules, guidance, consistency, quality gates, operational clarity"
- evoke architecture, specification systems, editorial precision, industrial design, and premium product branding

Color palette:
- dominant graphite, ink black, deep charcoal, mineral gray
- secondary warm off-white / bone / paper white
- one restrained accent only, such as muted brass, steel blue, or dark cobalt
- no rainbow palette
- no loud cyberpunk color combinations

Lighting:
- controlled studio lighting
- soft but directional key light
- precise shadow edges
- subtle reflections on glass/metal
- premium product photography mood, not fantasy glow

Texture and materials:
- anodized aluminum
- smoked glass
- ceramic matte surfaces
- dense archival paper
- micro-engraved lines
- subtle embossed marks
- extremely clean finish

Typography treatment:
- if typography is included, render "Agent Standards" as a refined modern wordmark
- use a neo-grotesk / Swiss editorial feel, similar in spirit to Neue Haas Grotesk, Söhne, or Suisse Intl
- typography should feel expensive, quiet, and authoritative
- avoid playful or futuristic gimmick fonts
- if text rendering is weak, prioritize the visual system and leave space for later type overlay

Style references:
- premium editorial technology campaign
- high-end industrial design identity
- luxury software launch artwork
- Swiss grid poster discipline
- sophisticated, understated, exacting, modern

Camera / framing:
- straight-on or very slightly elevated perspective
- medium focal length look
- minimal distortion
- sharp focus
- pristine edges

Mood:
- composed
- intelligent
- rigorous
- premium
- operational
- calm authority
- high trust

Absolutely avoid making it look like:
- a generic AI startup landing page
- a hacker poster
- a gaming splash screen
- an NFT render
- a SaaS dashboard screenshot
- a random abstract wallpaper

Render quality:
- ultra-clean
- high detail
- polished
- brand-system level intentionality
- premium materials
- cinematic but restrained
- suitable for README hero, launch post, website masthead, and social banner
```

## Negative Prompt

```text
purple neon, cyberpunk, glowing circuitry, robot face, humanoid robot, brain icon, hologram UI, HUD overlays, sci-fi city, childish mascot, cartoon style, anime, low-end startup illustration, flat generic vector blobs, cheap gradient mesh, oversaturated colors, rainbow accents, gamer aesthetic, hacker aesthetic, cliché AI imagery, stock SaaS artwork, cluttered composition, busy background, plastic materials, toy-like forms, soft blurry focus, distorted typography, noisy textures, excessive lens flare, floating random shapes, futuristic nonsense
```

## Optional Variants

### Variant A: README Hero

```text
Same concept, but optimized for a wide README / website masthead. Leave controlled negative space around the central system so a title can be overlaid cleanly if needed.
```

### Variant B: Square Social Cover

```text
Same concept, but optimized for a square launch card with slightly stronger central emphasis and clearer silhouette recognition at small size.
```

### Variant C: Emblem-First

```text
Same concept, but reduce environmental detail and focus on building a strong brand emblem from a core block plus modular extensions, keeping a premium industrial-editorial finish.
```

## Suggested Outputs

- `1792x1024` for website / README hero
- `1536x1536` for square social / cover art
- `1344x768` for lightweight preview

## Final Direction

If the model has to choose between:
- "more obviously AI"
- "more premium, structured, and standards-driven"

always choose the second.
